# TM Shewer Business Website

## Description
This is a professional business website created for TM Shewer. The website presents information about the company and the services it offers, including construction, catering, telecommunications, and logistics. The site is designed to be clear, informative, and easy to navigate.

## Features
- Company overview
- Construction services information
- Catering services information
- Telecommunications services information
- Logistics services information
- Contact details
- Clean and user-friendly layout

## Tools & Technologies
- HTML
- CSS
- Markdown
- GitHub Pages

## How to Use
1. Open the website in a web browser.
2. Navigate through the sections using the menu.
3. View detailed information about each service offered by TM Shewer.
4. Use the contact section for enquiries or service requests.

## Purpose of the Project
This project was created to demonstrate basic web development skills using HTML and CSS while building a professional multi-service business website and deploying it using GitHub Pages.
